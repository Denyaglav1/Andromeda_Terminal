# Push Changes to GitHub

Follow these steps to commit and push all current changes to the remote GitHub repository.

// turbo-all

1. Check current git status to see what has changed:
```bash
git status
```

2. Stage all changes:
```bash
git add .
```

3. Commit with a descriptive message (ask the user for a message or generate one based on changes):
```bash
git commit -m "<описание изменений>"
```

4. Push to the remote repository:
```bash
git push
```
